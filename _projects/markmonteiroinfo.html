This website branched off of a free base theme from startbootstrap.com and modified appropriately. A notable modification is the incorporation and modification of a vertical timeline from codyhouse.co.

The site is hosted using GitHub pages, which profides simple, free hosting for static websites using the Jekyll web framework. The collection and templating features of Jekyll are used to abstract the rendering of each project in the portfolio section.
