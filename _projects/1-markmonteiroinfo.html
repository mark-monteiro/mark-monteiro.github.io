This website branched off of a free base theme from startbootstrap.com. Notable modifications include the incorporation and modification of a vertical timeline from codyhouse.co, and using the collections and templating features of the Jekyll framework to abstract the rendering of each project in the portfolio section. The site is hosted using GitHub pages, which profides simple, free hosting for static websites using the Jekyll web framework.
