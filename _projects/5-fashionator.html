--- title: Fashionator id: fashionator developed: April 2014 technologies: - PHP - MySQL ---

The fashionator project was started to give users the ability to catalogue their clothes in order to buy and sell them, as well as recieve fashion tips from real experts. The website was developed using PHP, backed by a MySQL database.

In the end, the website was never completed and the project was abandoned in order to move on to more promising ideas.
